动态规划初体验
2019-04-07|Learning Notes

题目描述：

一个长度为N的序列，玩家每次只能从头部或尾部拿数字，不能从中间拿。拿走的数字依次从左到右排列在自己面前。拿完N个数字之后，游戏结束。此时N个数字在玩家面前组成一个新的排列，这个数列每相邻两个数字之差的绝对值之和为玩家最终得分。假设玩家前面的N个数字从左到右标号为 n1,n2,…nN，则最终得分S的计算方式如下：
S = abs(n1-n2) + abs(n2-n3) + … + abs(n(N-1) - nN).

请计算玩家在以上游戏规则中把所有数字拿完可以获得的最大得分。
